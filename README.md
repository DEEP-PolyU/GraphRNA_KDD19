# GraphRNA_KDD19
Graph Recurrent Networks with Attributed Random Walks

## Installation
- Requirements
1. numpy
2. scipy
3. sklearn
4. torch
- Usage
1. cd GraphRNA_KDD19
2. pip install -r requirements.txt
3. python GraphRNA_endtoend.py --gpu 0
